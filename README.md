# What's For Dinner?
A fullstack web application, where a user can look up recipes based on available ingredients. 
User can create an account, view the detailed recipe, and save favourite recipes. 

## Live demo:
(https://what-s-for-dinner.onrender.com/)

## Source Code: 
(https://github.com/bhatnagar-kritika/What-s-for-dinner-.git)

## Features
- Search recipes using available ingredients through spoonacular API to retrieve recipe instructions.
- User registration and login using JWT authentication, password hashed using bcrypt.
- Recipes can be added, removed, and viewed in favourites section.
- Recipe details cached in Mongodb to reduce repeated API calls.
- Responsive user interface built with Tailwind CSS.
